# VUR Website

Website resmi untuk **VUR (Void User Repository)** — indeks paket komunitas untuk Void Linux.
Website ini menyediakan cara cepat dan sederhana untuk mencari, melihat, dan mengakses paket yang tersedia di repository **VUR**.

VUR-Website : https://t4n-labs.github.io/vur-web

## Tentang VUR

**VUR (Void User Repository)** adalah repository komunitas yang menyediakan template paket tambahan untuk **Void Linux**.

Tujuan proyek ini:

* Mempermudah distribusi paket komunitas
* Menyediakan indeks paket yang mudah digunakan
* Mendukung ekosistem Void Linux

## Sumber Data Paket

Semua data paket berasal dari repository utama VUR:

https://github.com/T4n-Labs/vur

Setiap paket memiliki struktur:

```
category/package-name/template
```

Kategori yang tersedia:

* `core`
* `extra`
* `multilib`
